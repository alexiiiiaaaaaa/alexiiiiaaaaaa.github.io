Battleship Game

This project is an enhanced version of the classic Battleship game, where a player competes against a computer opponent. The game is set on a 10x10 grid, with rows labeled A-J and columns numbered 1-10. Unlike traditional versions, ships of varying sizes are automatically generated and placed on the grid, allowing the user to focus entirely on gameplay—finding and targeting the computer's hidden ships.

The game includes a user-friendly menu, offering options to start a new game or save progress. It features a graphical user interface (GUI) with dynamic animations and optional sound effects, which can be toggled on or off according to user preference. The animations adapt to the game’s outcome, with different visuals for hits and misses, adding an extra layer of engagement.

The program tracks the player's performance, including the total number of shots and successful hits. It provides instant feedback for each shot, letting the player know whether they’ve hit a ship or missed.

Developing this project helped me gain deeper insights into software development, particularly in building an interactive GUI and managing game logic for single-player experiences against AI. It also enhanced my understanding of event-driven programming and gave me valuable practice in designing a fully functional and responsive application.

Languages: Java

Lottery Simulator

Test Your Luck! - Number Matching Game

This project is a number-matching game that tests the user's luck. The program generates a set of unique numbers within a specified range and compares them with the user's input. It ensures that all numbers entered by the user are valid before proceeding with the comparison. Depending on the number of matches between the generated numbers and the user's input, the game produces a corresponding result. If there are no matches, the user receives a specific message, but if they match a certain number of numbers, they earn virtual rewards (coins). The program is equipped with automated tests to ensure its accuracy and reliability. This project showcases my skills in validation logic, random number generation, and implementing user feedback systems.

Languages: Java

College Network Simulator

College Network Topology Implementation Using Cisco Packet Tracer

This project involves the design and implementation of a comprehensive network topology for a college campus using Cisco Packet Tracer. The network includes an effective IP addressing scheme and various services like DHCP, DNS, and FTP servers to ensure smooth communication and data transfer. I configured routing protocols such as OSPF and EIGRP to ensure efficient data routing between different network segments, while VLANs were used to segregate network traffic, enhancing security and performance. This project gave me hands-on experience with network design, configuration, and troubleshooting in a simulated environment, providing valuable insights into real-world networking.

TOOLS: Cisco Packet Tracer

Hospital Scheduler (team project)

The project focuses on developing a scheduling system for a hospital. In the healthcare sector, various professionals such as doctors, nurses, and physiotherapists work together. Each healthcare professional has their own electronic diary, which allows them to record details of patient visits, such as consultations, examinations, treatments, and more.

Our team implemented various features, including a search functionality that allows users to find available dates and times across multiple professionals' diaries, making it easier to schedule patient appointments. Additionally, we developed a user-friendly graphical interface for the program, enhancing usability and accessibility.

This project is a collaborative effort, which has allowed us to deepen our understanding of Java and GUI development. Working as a team has also taught us how to effectively plan tasks, write clean code, and manage project timelines. Moreover, it has helped me improve my communication skills and time management, both essential for working in a team environment.

Languages: Java
